# AndroidVM Manager

Una aplicación móvil avanzada de máquina virtual para Android diseñada para ejecutar entornos ligeros como Android Go y ROMs personalizadas mediante arquitectura de contenedores.

## Características planificadas:
- Soporte para Android Go Edition y sistemas limpios.
- Gestor de imágenes de ROMs personalizadas.
- Aislamiento de procesos y optimización de recursos para dispositivos de gama baja.
